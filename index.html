  • .通配符

  • \d 数字：[0-9]

  • \D 非数字： [^0-9]

  • \w 单词字符：[a-zA-Z_0-9]

  • \W 非单词字符：[^\w]

    算法

  • [a-d[m-p]] a到 d 或 m 到 p：[a-dm-p]（并集）

  • [a-z&&[def]] d、e 或 f（交集） // 类似减法

    次数类

  • X? X，零次或一次 （不能是多次）

  • X* X，零次或多次 （除了0次）

  • X+ X，一次或多次 （拒绝0次）

  • X{n} 恰好 n 次

  • X{n,} 至少 n 次

  • X{n,m} 至少 n 次，但不超过 m 次
    （非捕获啥的有点蒙）

maven

学习的原因：项目构建过程包括【清理项目】→【编译项目】→【测试项目】→【生成测试报告】→【打包项目】→【部署项目】

理想的项目构建是高度自动化，跨平台，可重用的组件，标准化的，使用 maven就可以帮我们完成上述所说的项目构建过程。

本地仓库统一管理jar包，使jar包与项目分离，减轻项目体积
多个项目的时候就能体会到jar包的好处

优点：

jar包管理 也称为依赖管理
Maven由Java语言开发，可跨平台使用

--- Maven将大型项目按照模块拆分成若干块，不同团队来维护，之间用远程仓库 中小型项目：一般分为**web、service、dao**层 Dao层叫做数据访问层，data access object Service层用**接口**来定义

用终端跑起来
Cd到目录，之后输入 mvn tomcat:run
编译过后src目录层会多出target
Tip：如果本地仓库没有jar包，就会自动到互联网上下载

如果用maven，需要打包src文件和pom文件

Maven的三个仓库
本地仓库：自己维护
远程仓库：公司维护 私服
中央仓库：第三方团队维护 国内一般使用阿里云提供的（到mirror里面去配置，如果不配置，就会到maven官方网站去下载，不挂代理的话，会非常慢）（是啥？）

常用命令：

--- Mvn **clean** 是清除编译后的文件 Mvn **compile** 编译：只编译main目录下的代码，不编译test下的 Mvn **test-compile** 编译test下的代码 **Test** 是运行test下的代码 **Package** 打包,包名是由artifactID，version，packaging（war、jar包）来的 **Install** 是上传到本地仓库 **Tomcat:run** 一键构建项目

机器学习的概述：

  1. 对于某类任务 T 和性能度量 P，如果一个计算机程序在 T 上以 P 衡量的性能随着经验 E 而自我完善，那么我们称这个计算机程序在从经验 E 中学习。???
  2. 机器学习本质上属于应用统计学，更多地关注如何用计算机统计地估计复杂函数，不太关注为这些函数提供置信区间。

十大常用算法：

  1. 决策树（Decision Trees）

  2. 朴素贝叶斯分类 (Naive Bayes Classification) 概率论知识

    应用到

    • 一封电子邮件是否是垃圾邮件
    • 一篇文章应该分到科技、政治，还是体育类
    • 一段文字表达的是积极的情绪还是消极的情绪？
    • 人脸识别

  3. 普通最小二乘回归 (Ordinary Least Squares Regression)

  4. 逻辑回归 (Logistic Regression)

    应用到

    • 信用评估
    • 测量市场营销的成功度
    • 预测某个产品的收益
    • 特定的某天是否会发生地震

  5. 支持向量机 (Support Vector Machines)
    SVM 是一种二分算法

    应用到

    • 广告
    • 大规模图片分类
    • 基于图片的性别检测

  6. 集成方法 (Ensemble Methods)

  1. 聚类算法 (Clustering Algorithms) 神经网络 / 深度学习
  2. 主成分分析 (Principal Component Analysis)
  3. 奇异值分解 (Singular Value Decomposition)
  4. 独立成分分析 (Independent Component Analysis)

深度学习的概述：

机器学习，实现人工智能的方法；深度学习，实现机器学习的技术。

深度学习典型模型：

  1. 卷积神经网络模型
  2. 深度信任网络模型
  3. 堆栈自编码网络模型

数据科学

数据科学（英语：Data Science），又称资料科学，是一门利用数据学习知识的学科，其目标是通过从数据中提取出有价值的部分来生产数据产品。它结合了诸多领域中的理论和技术，包括应用数学、统计、模式识别、机器学习、数据可视化、数据仓库以及高性能计算。数据科学通过运用各种相关的数据来帮助非专业人士理解问题。 数据科学技术可以帮助我们如何正确的处理数据并协助我们在生物学、社会科学、人类学等领域进行研究调研。此外，数据科学也对商业竞争有极大的帮助。
