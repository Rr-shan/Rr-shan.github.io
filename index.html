聚类的评估指标
更新于 2019-12-06
  • 轮廓系数（Silhouette Coefficient）
  • Calinski-Harabaz 指数
  • 调整兰德系数（Adjusted Rand index，ARI）
  • 互信息（Adjusted Mutual Information，AMI）
  • V-measure
  • Fowlkes-Mallows Index（FMI）

轮廓系数（Silhouette Coefficient）

  • 内部指标
  • 选择合适的聚类数目
  • 根据折线图可直观的找到系数变化幅度最大的点，认为发生畸变幅度最大的点就是最好的聚类数目。

Calinski-Harabaz 指数

  • 内部指标
  • 选择合适的聚类数目，运算速度远高于轮廓系数
  • 当内部数据的协方差越小，类别之间的协方差越大，Calinski-Harabasz分数越高。 越大越好

调整兰德系数（Adjusted Rand index，ARI）

  • 外部指标
  • 用来衡量两个分布的吻合程度，取值范围[-1,1],数值越接近于1越好

互信息（Adjusted Mutual Information，AMI）

  • 外部指标
  • 用来衡量两个分布的吻合程度，取值范围[-1,1],数值越接近于1越好

V-measure

  • 外部指标
  • 同质性（homogeneity）：每个群集只包含单个类的成员。
  • 完整性（completeness）：给定类的所有成员都分配给同一个群集。
  • V-measure是两者的调和平均。V-measure取值范围为 [0,1]，越大越好，但当样本量较小或聚类数据较多的情况，推荐使用AMI和ARI。

Fowlkes-Mallows Index（FMI）

  • 外部指标
  • FMI是对聚类结果和真实值计算得到的召回率和精确率，进行几何平均的结果，取值范围为 [0,1]，越接近1越好。

一般情况下，主要是对无y值的数据进行聚类操作。如果在评价中用到外部指标，就需通过人工标注等方法获取y值，成本较高，因此内部指标的实际实用性更强。

MAGIC,DrImpute, ScImpute, SAVER, VIPER, and DCA

  1. MAGIC [18] focuses on cell/cell interactions to build a Markov transition matrix and smooth the data. 构建马尔科夫转移矩阵 平滑数据
  2. ScImpute [19] builds a LASSO regression model for each celland imputes them iteratively. LASSO回归模型，迭代进行估算
  3. SAVER [20] is a Bayesian—based model using various prior probability functions.
  4. DrImpute [21] is a clustering-based method and uses a consensus strategy: it estimates a value with several cluster priors or distance matrices and then imputes by aggregation. 聚类的方法进行估算
  5. VIPER is a recent published statistical method that looks at cell/cell interaction to fit a linear model for each cell.
  6. DCA builds an auto-encoder to model the genes distribution using a zero inflated negative binomial prior.
