Java

（2019.11.4~2019.11.10）

  • -

    上周：

  1. 机器学习大多数算法，深度学习入门（有些赶）
  2. 看论文（终于能好好学英语了！），学习写论文的方法，写了一篇博客

下周：

  1. 开始着手用Python实现机器学习的算法《机器学习实战》
  2. 期中复习
  3. 将上周没完成的继续完成

决定和一个同学找沙瀛老师学习科学研究的一套方法，如果学的好，希望自己能够主持自己感兴趣方面的课题。

(2019.10.22~2019.11.3)

    • 上周：

  1. 认真看了《Java编程思想》有关容器和集合的部分，没有理解完全，不过终于开始体会到读书的好处了。
  2. 在github上看了聊天器的项目，一部分涉及到了spring boot框架、数据库以及前后端结合的知识，深觉自己基础知识很不ok，决定恶补Java基础 // 他们写的聊天器不在我思路能够理顺的范围内
  3. C++题库题目总结了一部分，额外总结了动态规划和分治算法（不是很扎实）、机器学习学在比较基础的阶段。

下周：

  1. 开始用Java实现C++题库题目，疯狂恶补Java基础，开始按照leedcode上面的大纲继续总结基本题目。
  2. 机器学习算法努力向前推进吧

可能带有一些惰性，写算法题会写到比较难受的状态。

(2019.10.21~2019.10.27)

  • -

上周：

  • cf并没有到1500，但相对之前有推进
  • 《Hadoop大数据分析与挖掘实战》看了很多实例，暂时还不能上手
  • 通过看书，理解了部分Javabeans、反射机制、集合知识（对自己来说，可能看视频掌握的更快，但忘得也比较快，归根到底还是要多用）
  • 机器学习的算法已经入门，对整体【机器学习】有了一个比较清晰的结构 // 难能可贵

下周：

  • 暂时放下Hadoop的学习，学习机器学习的算法，为之后想要研究方向的打基础（看吴达恩机器学习的视频、看西瓜书和《机器学习实战》）// 不是很确定自己学习的进度
  • 因为get到了上一届C++的题库，想先拿题库的题目来练手、再用Java把C++题再实现一遍 // 估计要三周搞定，五六个晚上
  • 到github上多读别人的项目（Java），争取能做到有自己的分析和想法再总结写到博客

cf心得：越写到后面题目意思的理解倒是有些吃力了
尽管题目是按照一定梯度来写的，还是遇到了有些题目好AC，有些题目难以AC的情况，弄得自己有点弄不清自己的水平（总是还是菜鸡）
不打算继续往前推进了，准备分门别类总结一下。顺道再多了解一些数据结构，打算花两个夜晚整理。
并不打算以竞赛为主，所以不会太注重。总结完之后差不多算一个小尾声了。
